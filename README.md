# ESPSomfy-RTS with Homekit

## Note for this fork
This code has been created to merge the [ESPSomfy-RTS](https://github.com/rstrouse/ESPSomfy-RTS) controller with Homekit. This should allow the device to connect directly with homekit and expose
shades there. 

NOTE: THIS HAS BEEN CREATED WITH THE HELP OF AI.

For all technical details like how to build the appropriate device, please refer to [ESPSomfy-RTS](https://github.com/rstrouse/ESPSomfy-RTS). This project has followed the same approach. At the time of writing the ESP32S3+ has been used because of its big flash size.
